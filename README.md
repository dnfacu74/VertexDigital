## 🛠️ Estructura Técnica del Sitio

El sitio está desarrollado con tecnologías modernas enfocadas en la velocidad de carga y la accesibilidad (Core Web Vitals):

*   **HTML5 Semántico:** Estructura clara para mejor indexación por parte de Google.
*   **CSS3 Moderno:** Uso de variables CSS, Grid y Flexbox para un diseño totalmente responsive.
*   **Optimización de Imágenes:** Formatos ligeros y carga diferida (lazy loading).
*   **Accesibilidad (A11y):** Navegación por teclado, contraste AA y etiquetas ARIA.
*   **Schema.org:** Datos estructurados para que Google muestre información rica (Person, Organization, Service).
