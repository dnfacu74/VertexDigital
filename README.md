# 🚀 Vertex Studio - Sitio Web Corporativo

Sitio web oficial de **Vertex Studio**, agencia premium especializada en soluciones digitales integrales: diseño web, audio branding, producción audiovisual y marketing digital.

🌐 **URL Pública:** [https://www.vertexdigital.com.ar](https://www.vertexdigital.com.ar)

---

## 📋 Descripción del Proyecto

Plataforma corporativa estática desarrollada con estándares modernos de rendimiento, accesibilidad y SEO técnico. Diseñada para transmitir innovación, calidad premium y confianza, posicionando a Vertex Studio como referente tecnológico en Argentina.

### ✨ Características Principales
- Diseño responsive mobile-first con estética dark premium
- Reproductores de audio personalizados para demos de Audio Branding
- Galería portfolio con filtrado por categoría y lightbox
- Animaciones sutiles con Intersection Observer (sin librerías pesadas)
- Optimización Core Web Vitals: lazy loading, fuentes locales WOFF2, imágenes WebP/AVIF
- SEO técnico completo: Schema.org JSON-LD, Open Graph, sitemap XML, robots.txt
- Integración directa con WhatsApp Business API (+54 9 221 602 6192)
- Compatible con Chrome, Firefox, Safari, Edge
